Brain Tumor Classification

The main purpose of this project was to build a CNN model that would classify if subject has a tumor or not base on MRI scan. I used a Convnet architecture from scratch to train the model for this binary problem. I used accuracy as a metric to justify the model performance which can be defined as:
Accuracy = (Number of correctly predicted images / total number of tested images ) * 100%

A brain tumor occurs when abnormal cells form within the brain. There are two main types of tumors: cancerous (malignant) tumors and benign tumors. Cancerous tumors can be divided into primary tumors, which start within the brain, and secondary tumors, which have spread from elsewhere, known as brain metastasis tumors.

Data Set Description
The image data that was used for this problem is Brain MRI Images for Brain Tumor Detection.
It conists of MRI scans of two classes:
  1. NO - no tumor, encoded as 0
  2. YES - tumor, encoded as 1
Since I had small data set I used Data Augmentation to increase the size of training set.




This project was a combination of CNN model classification problem (to predict wheter the subject has brain tumor or not) & Computer Vision problem (to automate the process of brain cropping from MRI scans).
